# Data Ownership Rules

## Ownership Types
- Created By
- Assigned To
- Store Owned
- HQ Owned
- System Locked

## Rules
- Staff → own data only
- Store Manager → store data
- HQ → cross-store
- Superadmin → everything

## Immutable Data
- Financial records (after lock)
- Prescriptions (after validity)
- Audit logs (never editable)
