# Navigation Rules

## Role-Based Landing
- Staff → /dashboard/staff
- Store Manager → /dashboard/manager
- Area Manager → /dashboard/area-manager
- Admin → /dashboard/admin
- Superadmin → /dashboard/superadmin

## Multi-Role Users
- Must choose role on login
- Role switch logged in audit trail

## Visibility Rules
- Unauthorized actions are visible but disabled
- No hidden power actions
- All clicks logged
