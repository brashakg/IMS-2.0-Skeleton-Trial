# INCENTIVE ENGINE — IMS 2.0

## Purpose
Prevent incentive abuse while motivating performance.

---

## 1. INCENTIVE TYPES

- Individual sales
- Team sales
- Store targets
- Category incentives
- Special campaigns

---

## 2. INCENTIVE GOVERNANCE

- Created only by Superadmin
- Activated per:
  - Role
  - Store
  - Category
  - Time period

---

## 3. INCENTIVE LOCKING

- Incentives lock at payroll freeze
- No retroactive edits
- Exceptions require Superadmin override

---

## 4. ABUSE PREVENTION

- Linked to Discount Abuse Engine
- Linked to Return patterns
- Linked to Customer manipulation detection

Flagged incentives go into:
→ Review queue

---

END INCENTIVE ENGINE
