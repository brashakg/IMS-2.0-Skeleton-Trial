# RETURN & REFUND ABUSE — IMS 2.0

## Purpose
Prevent misuse of return policies.

---

## PATTERN 1 — HIGH RETURN RATE
Pattern:
- Same staff / store with frequent returns

Detection:
- Return ratio

Action:
- Admin insight

---

## PATTERN 2 — REFUND TIMING GAP
Pattern:
- Refund delayed intentionally

Detection:
- Return vs refund delta

Action:
- Finance task
