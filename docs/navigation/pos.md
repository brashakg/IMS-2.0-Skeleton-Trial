# POS NAVIGATION & ACCESS CONTROL

## Entry Points
- Dashboard → New Sale
- Orders → Create Order
- Customer → Create Order

---

## ROLE VISIBILITY

### Sales Staff
- Create Order
- View own orders
- Cannot override discounts
- Cannot edit prices

### Store Manager
- Approve limited discounts
- View store orders
- Cannot change MRP / Offer

### Area Manager
- Approve higher discounts
- Multi-store visibility

### Admin / Superadmin
- Full visibility
- Override pricing rules
- Audit access

---

## BUTTON STATE LOGIC

Buttons can be:
- Enabled
- Disabled (with tooltip reason)
- Approval-gated

No hidden actions.
