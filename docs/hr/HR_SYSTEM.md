## Purpose
To manage:
- Attendance
- Leaves
- Salary
- Incentives
- Advances
- Discipline

With full transparency to employees and strict control by management.

---

## 1. HR ACTORS

- Employee
- Store Manager
- Area Manager
- HR / Admin
- Superadmin

Each actor sees only what they are allowed to see.

---

## 2. EMPLOYEE PROFILE (SINGLE SOURCE OF TRUTH)

Each employee has:
- Personal details
- Assigned store(s)
- Role(s)
- Shift timings
- Weekly offs
- Leave policy
- Salary structure
- Incentive rules
- Discount authority (linked to POS)

Editable only via Superadmin / Admin approval.
