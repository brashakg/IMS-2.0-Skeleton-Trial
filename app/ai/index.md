# AI & Ask Intelligence Screens Index

This file represents the root entry for all AI and intelligence-related screens.

The AI layer provides:

- Read-only insights and recommendations
- Pattern detection across sales, inventory, HR, and operations
- Risk flags and anomaly detection
- Change proposals requiring Superadmin approval
- Natural language querying for Superadmin use only

AI has no direct execution authority.
Governance rules are defined in
`docs/SYSTEM_INTENT.md`.
