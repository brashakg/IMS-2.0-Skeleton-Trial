# Marketplace Screens Index

This file represents the root entry for all marketplace-related screens.

The Marketplace module governs:

- Product sync across Shopify, Amazon, and Flipkart
- Inventory availability and blocking rules
- Order ingestion and tracking
- SLA monitoring for dispatch and delivery
- Customer order status visibility
- Courier integration and intelligence

Marketplace intelligence feeds into Ask Intelligence.
All rules are defined in
`docs/SYSTEM_INTENT.md`.
