# Inventory & Stock Management

This module manages all inventory across HQ and stores.

Includes:
- HQ cataloging
- Store receiving & acceptance
- Escalation for mismatches
- Location assignment (counter/shelf/cabinet)
- Barcode printing (store-level)
- Transfers, quarantines, returns
- Vendor handling
- SLA tracking

Inventory is the backbone of IMS 2.0.
All rules are enforced strictly as per SYSTEM_INTENT.md.
