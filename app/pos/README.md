# Point of Sale (POS)

This module defines all Point of Sale behavior.

Scope includes:
- Sales flow
- Pricing logic (MRP / Offer Price / Discounts)
- Role-based overrides
- Gift cards
- Payments
- Customer handling
- Prescription attachment
- Barcode handling
- AI-assisted recommendations (read-only)

POS logic is strict, deterministic, and audited.
No shortcuts or silent overrides are allowed.

All workflows, edge cases, and simulations are defined in SYSTEM_INTENT.md.
